# space-test — Apprentissage du Néerlandais 🇳🇱

Projet personnel d'apprentissage de la langue néerlandaise, structuré pour
préparer le **CNaVT** (Certificat de Néerlandais comme Langue Étrangère).

## Profil de l'apprenant

| Élément | Valeur |
|---|---|
| Langue cible | Néerlandais |
| Niveau actuel (CECRL) | Débutant → début intermédiaire (A2/B1) |
| Objectif | Réussir le CNaVT |
| Rythme | 15 min / jour |
| Thèmes | Tous : professionnel, travail, quotidien, grammaire/conjugaison |
